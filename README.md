# 贪吃蛇游戏

一个经典的贪吃蛇网页游戏，支持速度调节和模拟广告展示。

## 功能特点

- 🐍 经典贪吃蛇游戏玩法
- 🎮 支持键盘控制（方向键和WASD）
- ⚡ 可调节游戏速度（50ms - 300ms）
- 🏆 实时得分和最高分记录
- ⏸️ 暂停/继续游戏功能
- 📱 响应式设计，适配不同屏幕
- 🎨 模拟广告展示（用于练习）

## 游戏操作

- **方向键** 或 **WASD**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **速度滑块**：调整游戏移动速度
- **重新开始按钮**：重置游戏

## 如何运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx http-server
   ```
3. 在浏览器中访问 `http://localhost:8000`

## 游戏规则

1. 控制蛇吃掉食物，蛇身会变长
2. 每吃一个食物得10分
3. 撞到墙壁或自己的身体，游戏结束
4. 最高分会保存在本地存储中
